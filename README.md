# My Dotfiles :floppy_disk:

Dotfiles for Arch Linux.

## Installation

To install, run:

```sh
wget -O - https://raw.githubusercontent.com/ribru17/.dotfiles/master/scripts/dotfiles_install.sh | bash
```
